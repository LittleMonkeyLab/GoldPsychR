Overview

In this lab, you’ll learn to compare three or more groups using Analysis of Variance (ANOVA). We’ll work through one-way and two-way designs, conduct post-hoc comparisons, calculate effect sizes, and write up results in APA format.

The Data Story

We’ve collected data from a Study Techniques experiment:

  • 120 university students participated in a memory test
  • Students used one of three study methods: Rereading, Self-Testing, or Elaboration
  • Half studied in the Morning, half in the Evening
  • Outcome: Number of items correctly recalled (0-30)

Our research questions: 1. Do the three study methods differ in effectiveness? 2. Does time of day affect memory performance? 3. Is there an interaction between study method and time of day?

Note

Learn More About Study Techniques

The testing effect is one of the most robust findings in cognitive psychology. Explore the research through the Goldsmiths Library.

Learning Objectives

By the end of this lab, you will be able to:

  • Explain when to use ANOVA instead of t-tests
  • Conduct a one-way ANOVA using aov()
  • Interpret F-ratios and p-values from ANOVA output
  • Conduct Tukey’s HSD post-hoc comparisons
  • Conduct a two-way (factorial) ANOVA
  • Identify and interpret main effects and interactions
  • Calculate and interpret eta-squared (η²)
  • Write APA-style ANOVA results

# Overview

In this lab, you'll learn to compare three or more groups using Analysis of Variance (ANOVA). We'll work through one-way and two-way designs, conduct post-hoc comparisons, calculate effect sizes, and write up results in APA format.

## The Data Story

We've collected data from a Study Techniques experiment:

- **120 university students** participated in a memory test
- Students used one of **three study methods**: Rereading, Self-Testing, or Elaboration
- Half studied in the **Morning**, half in the **Evening**
- Outcome: Number of items correctly recalled (0-30)

**Our research questions**:
1. Do the three study methods differ in effectiveness?
2. Does time of day affect memory performance?
3. Is there an interaction between study method and time of day?

::: {.callout-note}
![](/assets/images/svg/brain.svg){width=20 style="float: left; margin-right: 10px;"}

## Learn More About Study Techniques

The testing effect is one of the most robust findings in cognitive psychology. [Explore the research](https://librarysearch.gold.ac.uk/discovery/search?query=any,contains,testing%20effect%20memory&tab=Everything&search_scope=MyInst_and_CI&vid=44GOL_INST:44GOL_VU1&offset=0) through the Goldsmiths Library.
:::

## ![](/assets/images/svg/target.svg){width=24} Learning Objectives

By the end of this lab, you will be able to:

- Explain when to use ANOVA instead of t-tests
- Conduct a one-way ANOVA using `aov()`
- Interpret F-ratios and p-values from ANOVA output
- Conduct Tukey's HSD post-hoc comparisons
- Conduct a two-way (factorial) ANOVA
- Identify and interpret main effects and interactions
- Calculate and interpret eta-squared (η²)
- Write APA-style ANOVA results

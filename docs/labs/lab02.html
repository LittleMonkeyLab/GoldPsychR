# Overview

In this lab, you'll learn to compare two groups using t-tests. We'll test whether there are significant differences between conditions and calculate effect sizes.

## Learning Objectives

By the end of this lab, you will be able to:

- Run independent samples t-tests
- Run paired samples t-tests
- Calculate and interpret Cohen's d
- Create comparison visualisations
- Write APA-style results

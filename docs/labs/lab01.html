Overview

The Data Story

# Overview

In this lab, you'll learn to analyse Grit Scale data in R. We'll work through loading data, reverse-scoring questionnaire items, calculating subscales, creating visualisations, and writing up results.

## The Data Story

We've collected data using the Short Grit Scale (Grit-S):

- **120 university students** completed the 8-item Grit Scale
- Participants are either **First Year** or **Final Year** students
- The scale measures two subscales: Consistency of Interest and Perseverance of Effort

**Our research question**: Do grit scores differ between first-year and final-year students?

::: {.callout-note}
![](/assets/images/svg/brain.svg){width=20 style="float: left; margin-right: 10px;"}

## Learn More About the Grit Scale

Want to read the original research? [Access the source article](https://librarysearch.gold.ac.uk/permalink/f/igqbsh/TN_cdi_informaworld_taylorfrancis_310_1080_00223890802634290) through the Goldsmiths Library.
:::

## ![](/assets/images/svg/target.svg){width=24} Learning Objectives

By the end of this lab, you will be able to:

- Navigate RStudio and understand its four panes
- Load CSV data into R using `read_csv()`
- Reverse-score negatively worded questionnaire items (4 items: G1, G3, G5, G6)
- Calculate subscale scores (Consistency of Interest, Perseverance of Effort)
- Calculate descriptive statistics by group using `dplyr`
- Create boxplots using `ggplot2`
- Write an APA-style results paragraph

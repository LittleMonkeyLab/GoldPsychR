## Usage

### HTML / Moodle
```html
<img src="icons/svg/slides.svg" alt="Slides" width="20" height="20">
<img src="icons/png/slides.png" alt="Slides" width="20" height="20">
```

### Quarto / RMarkdown
```markdown
![](icons/svg/slides.svg){width=20}
![](icons/png/slides.png){width=20}
```

### Format Recommendations

| Format | Use |
|--------|-----|
| HTML / Moodle | SVG or PNG |
| RevealJS | SVG |
| PDF | PNG |
| Word | PNG |

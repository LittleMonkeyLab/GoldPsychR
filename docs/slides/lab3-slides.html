The Dataset: Study Techniques

Research Question: Do different study methods affect memory recall?

  • 120 university students
  • 3 study methods: Rereading, Self-Testing, Elaboration
  • 2 time periods: Morning or Evening
  • Outcome: Items recalled (0-30)

Why Not Multiple t-tests?

Comparing 3 groups means 3 pairwise comparisons:

  • Group 1 vs Group 2
  • Group 1 vs Group 3
  • Group 2 vs Group 3

Problem: Each test at Œ± = .05 inflates overall error rate!

\[\text{Family-wise error} = 1 - (1 - .05)^3 = .14\]

ANOVA controls this!

Part 1: One-Way ANOVA

The Logic of ANOVA

ANOVA compares between-group variance to within-group variance

\[F = \frac{\text{Between-group variance}}{\text{Within-group variance}}\]

  • Large F ‚Üí Groups differ more than expected by chance
  • Small F ‚Üí Differences could be due to random variation

Part 2: Post-Hoc Tests

Why Post-Hoc Tests?

ANOVA tells us: ‚ÄúAt least one group differs‚Äù

But we need to know WHICH groups differ!

Post-hoc tests compare all pairs while controlling Type I error

Tukey‚Äôs HSD

HSD = ‚ÄúHonest Significant Difference‚Äù

# Run Tukey's test

Part 3: Two-Way ANOVA

Factorial Designs

When you have two factors:

  • Factor A: Study Method (3 levels)
  • Factor B: Time of Day (2 levels)

Design: 3 √ó 2 = 6 cells

What Can We Test?

Three questions in one analysis:

  1. Main effect of Method: Do methods differ (averaging over time)?
  2. Main effect of Time: Does time matter (averaging over method)?
  3. Interaction: Does the effect of method depend on time?

Running Two-Way ANOVA

# Use * to include interaction
model2 <- aov(recall ~ method * time_of_day, data = study_data)

summary(model2)

Two-Way ANOVA Output

                    Df Sum Sq Mean Sq F value   Pr(>F)
method               2  899.6  449.80  23.42 2.1e-09 ***
time_of_day          1   62.8   62.80   3.27   0.073
method:time_of_day   2   11.2    5.60   0.29   0.748
Residuals          114 2188.4   19.20

Interpreting the Results

Effect F p Interpretation
Method 23.42 <.001 ‚úì Significant main effect
Time 3.27 .073 ‚úó Not significant
Interaction 0.29 .748 ‚úó Not significant

Method matters, but time of day doesn‚Äôt, and the pattern is consistent across times.

What is an Interaction?

No interaction: The effect of A is the same at all levels of B

Interaction present: The effect of A depends on the level of B

Example with interaction: - Self-testing works great in the morning - Self-testing is worse in the evening - The effect of method changes with time

Interaction Plots

Part 4: Effect Sizes

Beyond p-values

A significant F tells us groups differ.

But how much do they differ?

Eta-squared (Œ∑¬≤) = proportion of variance explained

Calculating Eta-Squared

\[\eta^2 = \frac{SS_{effect}}{SS_{total}}\]

# Extract from summary
s <- summary(model1)

ss_method <- s[[1]]["method", "Sum Sq"]
ss_residual <- s[[1]]["Residuals", "Sum Sq"]
ss_total <- ss_method + ss_residual

eta_sq <- ss_method / ss_total
cat("Œ∑¬≤ =", round(eta_sq, 3))

Cohen‚Äôs Guidelines for Œ∑¬≤

Œ∑¬≤ Interpretation
0.01 Small
0.06 Medium
0.14 Large

Our Œ∑¬≤ ‚âà 0.29 ‚Üí Large effect

Study method explains ~29% of variance in recall!

Partial Eta-Squared

For factorial designs, use partial eta-squared:

\[\eta^2_p = \frac{SS_{effect}}{SS_{effect} + SS_{error}}\]

Controls for other factors in the model.

Part 5: APA Reporting

One-Way ANOVA Template

A one-way ANOVA revealed a [significant/non-significant] effect of [factor] on [DV], F(df1, df2) = F-value, p = p-value, Œ∑¬≤ = effect-size.

Example Write-Up

A one-way ANOVA revealed a significant effect of study method on recall performance, F(2, 117) = 24.10, p < .001, Œ∑¬≤ = .29. Post-hoc comparisons using Tukey‚Äôs HSD indicated that self-testing (M = 21.5, SD = 4.2) and elaboration (M = 19.7, SD = 4.5) both resulted in significantly higher recall than rereading (M = 14.9, SD = 4.1), ps < .001. Self-testing and elaboration did not differ significantly (p = .22).

Two-Way ANOVA Template

A 3 √ó 2 ANOVA examined the effects of [factor A] and [factor B] on [DV]. There was a significant main effect of [factor A], F(df1, df2) = F, p = p, Œ∑¬≤_p = value. The main effect of [factor B] was [not] significant. The interaction was [not] significant.

Common Reporting Mistakes

  1. Forgetting effect sizes - Always report Œ∑¬≤ or Œ∑¬≤_p
  2. Wrong df order - It‚Äôs F(between, within)
  3. Not rounding appropriately - F to 2 decimal places
  4. Missing post-hocs - Need them to specify which groups differ
  5. Over-interpreting non-significant effects
